Progetto per l'esame di ISPW 2025-2026 del corso di ingegneria informatica di Università degli di studi di Roma Tor vergata. 

Per accedere alla modalità cli passare il flag --cli all'eseguibile
Per accedere alla modalità demo in cui i dati non sono salvati nella persistenza, passare il flag --demo

Il programma offre una doppia modalità di persistenza, FS/Json e Db relazionale.
Di default il programma usa il database. 
Per usare il FS/Json passare il flag --fs all'eseguibile.
